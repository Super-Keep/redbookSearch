⚙️ 服务配置

🔴 小红书 Cookie 配置

获取步骤：

  1. 在电脑浏览器打开 www.xiaohongshu.com 并登录你的账号
  2. 按 F12 打开开发者工具
  3. 切换到 网络 (Network) 标签页
  4. 点击 Fetch/XHR 过滤器
  5. 在页面上随便操作一下（比如搜索），让网络面板出现请求
  6. 点击任意一个请求，在右侧 请求标头 (Request Headers) 中找到 cookie 字段
  7. 复制整个 cookie 值粘贴到下方输入框

⚠️ Cookie 有效期约 7-30 天，过期后需重新获取。如果某个请求没有 cookie 字段，换一个请求看。

💬 微信公众号 API Key 配置

前置条件：你需要有一个微信公众号。如果还没有，请先创建：

  1. 访问 微信公众平台，点击"立即注册"
  2. 选择"订阅号"类型（个人即可注册）
  3. 按提示填写邮箱、密码、个人信息完成注册

获取 API Key 步骤：

  1. 访问 down.mptext.top
  2. 使用你的公众号扫码登录
  3. 登录后在页面上找到并复制你的 API Key

⚠️ API Key 长期有效，一般不需要频繁更换
